🚀 Day 13 of 30 Days of Vibe Coding: Aura Quotes! ✨

For Day 13, I wanted to build something simple but execute it with the highest level of polish possible. I built a Daily Quote Generator, but I designed it to feel like a premium, luxury digital experience.

Meet Aura Quotes 🌌 — A minimalist, cinematic source of daily inspiration.

Here is what makes this project special: 👇

📚 The Data:
Instead of hardcoding a few quotes, the app connects to a free API on load, pulling in a massive dataset of over 1,400 quotes. It filters them for the perfect length and instantly serves them up without any loading spinners between clicks.

🎭 Cinematic Animations:
I used Framer Motion to create a "blur-fade" transition. When you click "Inspire Me", the old quote softly blurs and fades out, and the new one elegantly fades in. It feels incredibly smooth.

🌌 Premium Aesthetic:
I went all-in on the design:
- A custom animated mesh gradient background that slowly pulses.
- An SVG noise overlay to give the screen a physical, textured feel.
- Glassmorphic buttons with subtle hover shimmers.
- Elegant typography pairing `Cinzel` (serif) with `Montserrat` (sans-serif).

🛠️ The Tech Stack:
- React 19 & Vite
- Tailwind CSS 4
- Framer Motion
- DummyJSON Quotes API
